       Attendance Tracker Program
---------------------------------------------
This Python program allows users to record student attendance along with their arrival time.
It stores attendance in a dictionary, display attendance summary, calculate absentees,
and also exports the record the to a CSV file for easy sharing and storage.
The program ensures data accuracy through validation and provides a clean summary of
present students with their arrival times

      Features
-----------------
- Add multiple students at once.
- Prevents incorrect or duplicate entries.
- Generates formatted attendance report.
- Saves attendance to CSV
- User-friendly command-line interface
-  Timestamped attendance using Python’s `datetime` 
- Simple, lightweight, and beginner-friendly 
   
    Overview
-----------------
This program helps in:
- Recording class attendance interactively. 
- Validating all inputs (like student count and names). 
- Storing each student's name with their arrival time. 
- Displaying attendance summary for the day.
